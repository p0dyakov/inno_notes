9. Average Value of a Function, Integration Applications

Author

Zakhar Podyakov

Published

September 19, 2025

1. Summary

1.1 The Problem of Continuous Averages
1.1.1 Finite vs. Continuous Averages

Typically, to find the average of a set of numbers, you sum them up and divide by the count of numbers. For example, the average of \(\{1, 4, 2, 5\}\) is \((1+4+2+5)/4 = 3\).

However, this method fails for a continuous variable described by a function, like \(f(x) = \sin(x)\) over the interval \([0, \pi]\). In this interval, there are infinitely many values of \(\sin(x)\). You cannot simply sum up an infinite number of values and divide by infinity; the concept is not well-defined.

1.1.2 Approximating the Average

To solve this, we can start by approximating. Instead of using all infinitely many values, we can take a finite, representative sample.

  1. Sample Points: Choose a large number of evenly spaced points within the interval \([0, \pi]\).
  2. Calculate Heights: Find the value (height) of the function, \(\sin(x)\), at each of these sample points.
  3. Compute Finite Average: Add up these heights and divide by the number of points sampled.

As we increase the number of sample points, making them closer together, this approximation should get closer to the true average value of the continuous function.

1.2 Connecting Averages to Integrals
1.2.1 From Finite Sum to Riemann Sum

The process of approximating an average leads directly to the concept of an integral. Letâ€™s analyze the formula for our finite approximation: \[ \text{Average} \approx \frac{\sum \sin(x_i)}{\text{Number of samples}} \] The number of samples can be expressed using the length of the interval and the distance between samples, which weâ€™ll call \(dx\). \[ \text{Number of samples} \approx \frac{\text{Length of Interval}}{dx} = \frac{\pi - 0}{dx} = \frac{\pi}{dx} \] Substituting this back into our average approximation: \[ \text{Average} \approx \frac{\sum \sin(x_i)}{\pi / dx} = \frac{1}{\pi} \sum \sin(x_i)dx \] This expression, \(\sum \sin(x_i)dx\), is a Riemann sum. It represents the sum of the areas of many thin rectangles under the curve of \(\sin(x)\).

1.2.2 The Limit and the Definite Integral

To get the exact average value, we let the number of samples approach infinity, which means the width of our rectangles, \(dx\), approaches zero. In the language of calculus, we take the limit as \(dx \to 0\). This transforms the Riemann sum into a definite integral.

The average height of \(\sin(x)\) from \(0\) to \(\pi\) is: \[ \text{Average Value} = \frac{1}{\pi} \int_{0}^{\pi} \sin(x) \,dx \] This gives us a general formula for the average value of any function.

1.3 The Average Value Formula
1.3.1 General Formula

The average value of a continuous function \(f(x)\) on a closed interval \([a, b]\) is given by: \[ \text{Average Value} = \frac{1}{b-a} \int_{a}^{b} f(x) \,dx \] This formula can be interpreted geometrically. The integral \(\int_{a}^{b} f(x) \,dx\) represents the signed area under the curve of \(f(x)\) from \(a\) to \(b\). The term \(b-a\) is the width of the interval.

Therefore, the formula is conceptually: \[ \text{Average Height} = \frac{\text{Area}}{\text{Width}} \] This means the average value is the height of a rectangle that has the same width (\(b-a\)) and the same area as the region under the curve of \(f(x)\).

1.3.2 Example Calculation: \(\sin(x)\)

To find the average value of \(\sin(x)\) on \([0, \pi]\): 1. Find the Antiderivative: The antiderivative of \(\sin(x)\) is \(-\cos(x)\). 2. Evaluate the Integral: Using the Fundamental Theorem of Calculus: \[ \int_{0}^{\pi} \sin(x) \,dx = [-\cos(x)]_{0}^{\pi} = (-\cos(\pi)) - (-\cos(0)) = (-(-1)) - (-1) = 1 + 1 = 2 \] 3. Apply the Average Value Formula: \[ \text{Average Value} = \frac{1}{\pi - 0} \int_{0}^{\pi} \sin(x) \,dx = \frac{1}{\pi}(2) = \frac{2}{\pi} \] The average height of the \(\sin(x)\) curve between \(0\) and \(\pi\) is \(\frac{2}{\pi} \approx 0.637\).

1.4 An Alternate Perspective: The Fundamental Theorem

The average value formula provides another way to understand the Fundamental Theorem of Calculus. \[ \text{Average Value} = \frac{1}{b-a} \int_{a}^{b} f(x) \,dx = \frac{F(b) - F(a)}{b-a} \] Where \(F(x)\) is the antiderivative of \(f(x)\).

The expression \(\frac{F(b) - F(a)}{b-a}\) is the definition of the average rate of change (or average slope) of the function \(F(x)\) over the interval \([a, b]\).

Since \(f(x)\) is the derivative (the instantaneous rate of change) of \(F(x)\), this reveals a profound connection: * The average value of a function \(f(x)\) over an interval is equal to the average slope of its antiderivative \(F(x)\) over that same interval.

2. Definitions

  • Continuous Variable: A variable that can take on any value within a given interval, as opposed to a discrete variable which can only take on specific, separate values.
  • Average Value of a Function: A concept that extends the arithmetic mean to continuous functions. It represents the height of a rectangle with the same base and area as the region under the functionâ€™s curve over that interval.
  • Antiderivative: A function \(F(x)\) is an antiderivative of a function \(f(x)\) if \(F'(x) = f(x)\). It is the reverse process of differentiation.
  • Riemann Sum: An approximation of the area under a curve, calculated by summing the areas of a finite number of simple shapes, typically rectangles.

3. Formulas

  • Average of a Finite Set: \(\text{Average} = \frac{x_1 + x_2 + \dots + x_n}{n}\)
  • Average Value of a Continuous Function: \(\text{Average} = \frac{1}{b-a} \int_{a}^{b} f(x) \,dx\)
  • Geometric Interpretation: \(\text{Average Height} = \frac{\text{Area Under Curve}}{\text{Width of Interval}}\)
  • Antiderivative Form (Mean Value Theorem for Integrals): \(\text{Average} = \frac{F(b) - F(a)}{b-a}\), where \(F'(x) = f(x)\).

4. Mistakes

  • Forgetting to divide by the interval length: Calculating only \(\int_{a}^{b} f(x) \,dx\) gives the total signed area under the curve, not the average value. You must divide this area by the width of the interval, \(b-a\).
  • Assuming the average value is simply the midpoint of the functionâ€™s values at the endpoints: Calculating \(\frac{f(a) + f(b)}{2}\) is incorrect for most functions. This method only works for straight lines.
  • Using an incorrect interval width: The denominator must be the total length of the interval, calculated as \(b-a\). A common mistake is using just \(b\) or forgetting to subtract.
  • Errors in finding the antiderivative: The entire calculation depends on correctly identifying the antiderivative of \(f(x)\) to evaluate the definite integral.

5. Examples

5.1. Average Value of a Parabola
Question: Find the average value of the function \(f(x) = x^2\) on the interval \([0, 3]\).
Click to see the solution
  1. Set up the formula: The interval is \([0, 3]\), so \(a=0\) and \(b=3\). The width is \(b-a=3\). \[ \text{Average} = \frac{1}{3-0} \int_{0}^{3} x^2 \,dx \]
  2. Find the antiderivative: The antiderivative of \(x^2\) is \(\frac{x^3}{3}\).
  3. Evaluate the definite integral: \[ \int_{0}^{3} x^2 \,dx = \left[ \frac{x^3}{3} \right]_{0}^{3} = \frac{3^3}{3} - \frac{0^3}{3} = \frac{27}{3} - 0 = 9 \]
  4. Calculate the average value: \[ \text{Average} = \frac{1}{3} \times 9 = 3 \]
Answer: The average value of \(f(x) = x^2\) on \([0, 3]\) is 3.
5.2. Average Temperature
Question: The temperature in degrees Celsius over a 12-hour period starting at noon is modeled by the function \(T(t) = 20 + 5\sin(\frac{\pi t}{12})\), where \(t\) is the number of hours past noon. What is the average temperature between noon (\(t=0\)) and midnight (\(t=12\))?
Click to see the solution
  1. Set up the formula: The interval is \([0, 12]\), so \(a=0\), \(b=12\), and the width is \(b-a=12\). \[ \text{Average Temp} = \frac{1}{12-0} \int_{0}^{12} \left(20 + 5\sin\left(\frac{\pi t}{12}\right)\right) \,dt \]
  2. Find the antiderivative:
    • The antiderivative of \(20\) is \(20t\).
    • The antiderivative of \(5\sin(\frac{\pi t}{12})\) is \(5 \left(-\frac{12}{\pi}\cos\left(\frac{\pi t}{12}\right)\right) = -\frac{60}{\pi}\cos\left(\frac{\pi t}{12}\right)\).
    • The full antiderivative is \(20t - \frac{60}{\pi}\cos\left(\frac{\pi t}{12}\right)\).
  3. Evaluate the definite integral: \[ \int_{0}^{12} T(t) \,dt = \left[ 20t - \frac{60}{\pi}\cos\left(\frac{\pi t}{12}\right) \right]_{0}^{12} \] \[ = \left(20(12) - \frac{60}{\pi}\cos(\pi)\right) - \left(20(0) - \frac{60}{\pi}\cos(0)\right) \] \[ = \left(240 - \frac{60}{\pi}(-1)\right) - \left(0 - \frac{60}{\pi}(1)\right) = 240 + \frac{60}{\pi} + \frac{60}{\pi} = 240 + \frac{120}{\pi} \]
  4. Calculate the average value: \[ \text{Average Temp} = \frac{1}{12} \left(240 + \frac{120}{\pi}\right) = 20 + \frac{10}{\pi} \approx 23.18 \]
Answer: The average temperature is \(20 + \frac{10}{\pi} \approx 23.18^\circ\)C.
5.3. Average Value of a Linear Function
Question: Find the average value of \(f(x) = 2x + 1\) on the interval \([1, 5]\).
Click to see the solution
  1. Set up the formula: The interval is \([1, 5]\), so \(a=1\), \(b=5\), and the width is \(b-a=4\). \[ \text{Average} = \frac{1}{4} \int_{1}^{5} (2x + 1) \,dx \]
  2. Find the antiderivative: The antiderivative of \(2x + 1\) is \(x^2 + x\).
  3. Evaluate the definite integral: \[ \int_{1}^{5} (2x + 1) \,dx = [x^2 + x]_{1}^{5} = (5^2 + 5) - (1^2 + 1) = (25+5) - (1+1) = 30 - 2 = 28 \]
  4. Calculate the average value: \[ \text{Average} = \frac{1}{4} \times 28 = 7 \] Note: For a linear function, the average value is simply the value at the midpoint of the interval. The midpoint is \((1+5)/2=3\), and \(f(3) = 2(3)+1 = 7\).
Answer: The average value is 7.
5.4. Average Value with a Negative Area
Question: Find the average value of \(f(x) = x^3 - 3x\) on the interval \([-2, 2]\).
Click to see the solution
  1. Set up the formula: The interval is \([-2, 2]\), so \(a=-2\), \(b=2\), and the width is \(b-a=4\). \[ \text{Average} = \frac{1}{4} \int_{-2}^{2} (x^3 - 3x) \,dx \]
  2. Find the antiderivative: The antiderivative of \(x^3 - 3x\) is \(\frac{x^4}{4} - \frac{3x^2}{2}\).
  3. Evaluate the definite integral: \[ \int_{-2}^{2} (x^3 - 3x) \,dx = \left[\frac{x^4}{4} - \frac{3x^2}{2}\right]_{-2}^{2} \] \[ = \left(\frac{2^4}{4} - \frac{3(2)^2}{2}\right) - \left(\frac{(-2)^4}{4} - \frac{3(-2)^2}{2}\right) \] \[ = \left(\frac{16}{4} - \frac{12}{2}\right) - \left(\frac{16}{4} - \frac{12}{2}\right) = (4-6) - (4-6) = -2 - (-2) = 0 \] Note: This is an odd function integrated over a symmetric interval, so the result is 0.
  4. Calculate the average value: \[ \text{Average} = \frac{1}{4} \times 0 = 0 \]
Answer: The average value is 0.
5.5. Average Value of a Constant Function
Question: Find the average value of \(f(x) = 5\) on the interval \([2, 10]\).
Click to see the solution
  1. Set up the formula: The interval is \([2, 10]\), so \(a=2\), \(b=10\), and the width is \(b-a=8\). \[ \text{Average} = \frac{1}{8} \int_{2}^{10} 5 \,dx \]
  2. Find the antiderivative: The antiderivative of \(5\) is \(5x\).
  3. Evaluate the definite integral: \[ \int_{2}^{10} 5 \,dx = [5x]_{2}^{10} = 5(10) - 5(2) = 50 - 10 = 40 \]
  4. Calculate the average value: \[ \text{Average} = \frac{1}{8} \times 40 = 5 \] Note: This makes intuitive sense. The average value of a constant function is the constant itself.
Answer: The average value is 5.
5.6. Average Value of \(e^x\)
Question: Find the average value of \(f(x) = e^x\) on the interval \([0, 2]\).
Click to see the solution
  1. Set up the formula: The interval is \([0, 2]\), so \(a=0\), \(b=2\), and the width is \(b-a=2\). \[ \text{Average} = \frac{1}{2} \int_{0}^{2} e^x \,dx \]
  2. Find the antiderivative: The antiderivative of \(e^x\) is \(e^x\).
  3. Evaluate the definite integral: \[ \int_{0}^{2} e^x \,dx = [e^x]_{0}^{2} = e^2 - e^0 = e^2 - 1 \]
  4. Calculate the average value: \[ \text{Average} = \frac{1}{2} (e^2 - 1) \]
Answer: The average value is \(\frac{e^2 - 1}{2}\).
5.7. Average Value of \(\frac{1}{x}\)
Question: Find the average value of \(f(x) = \frac{1}{x}\) on the interval \([1, e]\).
Click to see the solution
  1. Set up the formula: The interval is \([1, e]\), so \(a=1\), \(b=e\), and the width is \(b-a=e-1\). \[ \text{Average} = \frac{1}{e-1} \int_{1}^{e} \frac{1}{x} \,dx \]
  2. Find the antiderivative: The antiderivative of \(\frac{1}{x}\) is \(\ln|x|\).
  3. Evaluate the definite integral: \[ \int_{1}^{e} \frac{1}{x} \,dx = [\ln|x|]_{1}^{e} = \ln(e) - \ln(1) = 1 - 0 = 1 \]
  4. Calculate the average value: \[ \text{Average} = \frac{1}{e-1} \times 1 = \frac{1}{e-1} \]
Answer: The average value is \(\frac{1}{e-1}\).
